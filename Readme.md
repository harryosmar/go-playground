# Go Playground

- [Concurrency](https://github.com/harryosmar/go-playground/blob/master/concurrency.md)

## Requirements

- [install Go](https://golang.org/doc/install)


## Getting started

```
git clone git@github.com:harryosmar/go-playground.git

cd go-playground

cp .env.example .env

go install

docker-compose up -d

make engine

./engine
```
